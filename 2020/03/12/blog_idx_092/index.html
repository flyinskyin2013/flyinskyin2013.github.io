    解决方案

      我看了网上的大部分方案都是说python版本过多的问题导致的。怎么说呢？原因确实可以这样说版本错误，因为你自己编译的版本和系统直接安装的版本编译参数是不一样的，所以有这个错。

    #重新编译python即可
./configure --prefix=/usr/local/python3 --enable-shared --with-fpectl   
make -j16
sudo make install

      出来啦，然后解决问题，完结散花。
